# Component 4: Application Tools

*Everything needed to submit strong applications.*

---

## Purpose

Application Tools help users create and submit job applications efficiently. From tailored resumes to autofill, these tools reduce friction and improve quality.

---

## Subcomponents

| Subcomponent | Description |
|--------------|-------------|
| [Resume Tailoring](./resume-tailoring/) | Customize resume per job based on ATS requirements |
| [Cover Letter](./cover-letter/) | AI-generated, job-specific cover letters |
| [CoPilot](./copilot/) | Chrome extension for autofill (Starter+ only, desktop) |
| [References](./references/) | Management and preparation of references |

---

## How It Connects

- Uses **Career Profile** as the source of user data
- Uses **Job Discovery** to understand job requirements
- Uses **Job Pockets** for tailored recommendations
- Integrates with **Planning & Support** for task tracking
