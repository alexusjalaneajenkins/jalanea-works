# Component 3: Job Pockets

*AI-powered briefings tailored to each job type.*

---

## Purpose

Job Pockets are comprehensive briefings that prepare users to apply for specific jobs. Each pocket type is tailored to a different job-hunting mode (Survival, Bridge, Career).

---

## Subcomponents

| Subcomponent | Description |
|--------------|-------------|
| [Basic Pocket](./basic-pocket/) | For Survival Jobs — clarity + tailored resume + cover letter |
| [Advanced Pocket](./advanced-pocket/) | For Bridge Jobs — strategic rebranding + skill bridging |
| [Professional Pocket](./professional-pocket/) | For Career Jobs — power skills + professional branding |

---

## Cost Philosophy

- Code it ourselves if possible
- Use AI only when necessary
- Keep costs low while maintaining quality

---

## How It Connects

- Uses **Career Profile** to personalize recommendations
- Uses **Job Discovery** data for job-specific insights
- Powers **Application Tools** with tailored content
- Generates tasks for **Planning & Support**
