# Component 5: Planning & Support

*Keeping users on track and informed.*

---

## Purpose

Planning & Support helps users stay motivated and make progress every day. It combines scheduling, task management, coaching, and education into a support system.

---

## Subcomponents

| Subcomponent | Description |
|--------------|-------------|
| [Calendar](./calendar/) | Realistic scheduling based on user availability |
| [Tasks & Microtasks](./tasks-microtasks/) | Daily actions with fallback 2-5 minute mini-tasks |
| [AI Coach](./ai-coach/) | Real questions, real answers for any job-hunting mode |
| [Education Hub](./education-hub/) | Definitions, components, types for job-hunting concepts |

---

## How It Connects

- Uses **Career Profile** to understand user schedule and preferences
- Uses **Job Discovery** to prioritize which jobs to focus on
- Uses **Job Pockets** to generate specific tasks
- Supports **Application Tools** by scheduling application activities
