# Component 1: Career Profile

*Understanding who the user is and what they bring to the table.*

---

## Purpose

The Career Profile component captures everything about the user that's relevant to their job search. This is the foundation that powers all other components.

---

## Subcomponents

| Subcomponent | Description |
|--------------|-------------|
| [Master Resume](./master-resume/) | Everything about the user (experience, education, credentials, skills, projects) |
| [Onboarding](./onboarding/) | Initial data gathering and self-assessment |
| [Job Type Selection](./job-type-selection/) | Survival / Bridge / Career mode |
| [Schedule & Availability](./schedule-availability/) | When and where they can work |

---

## How It Connects

- **Job Discovery** uses Career Profile to match users to jobs
- **Job Pockets** use Career Profile to tailor recommendations
- **Application Tools** use Career Profile to generate resumes and cover letters
- **Planning & Support** uses Career Profile to schedule realistic tasks
