# JalaneaWorks Future Features Roadmap

This document tracks planned features and enhancements for future development.

---

## Hiring Intel System

### Phase 1: Core Carousel (Current)
- [x] Seasonal hiring intelligence carousel on dashboard
- [x] Cards that update based on current date/month
- [x] Card types: Hot Now, Coming Up, Warning, Tip, Spotlight
- [x] Basic CTA actions (search, link)

### Phase 2: Coach Integration
- [ ] Cards can open Coach with pre-filled context/prompt
- [ ] "Ask Coach about this" CTA action
- [ ] Coach responses tailored to hiring season insights
- [ ] Call scripts available via Coach

### Phase 3: Content Hub
- [ ] Blog/Article system for hiring intel deep dives
- [ ] Cards link to relevant articles
- [ ] Video content integration (YouTube embeds)
- [ ] "Learn More" expanded content per card

### Phase 4: AI-Generated Daily Briefs
- [ ] System that scrapes job market news
- [ ] AI synthesizes news into daily hiring brief
- [ ] Auto-published blog posts
- [ ] Personalized briefs based on user's target sectors

### Phase 5: Smart Features
- [ ] User location-aware seasonal adjustments (e.g., spring starts earlier in Florida)
- [ ] "Remind me when X season starts" notifications
- [ ] Personalized recommendations based on user's industry preferences
- [ ] Integration with Job Pocket analysis
